Play Station

This play station is for general use for all CSE AI/ML Students. This play station was made for learning HTML in a unique and playful way. Play and learn HTML basics.

How to use this play station ? Below are the steps to use it efficiently.

  1. Open Virtual Studio Code
  2. Create a new text file.
  3. Select the language HTML.
  4. Start the code with declaring the version of the HTML.
  5. In parallel , open Show preview for observing the changes occuring as you type the code.
  6. Then, in the last click the option open in browser.

Some important things to remember.

  • Create a file in advance to add some images using link method.
  • One may use the Notepad for writing the code, but the preffered is VS Code.
  • HTML version 5 is required for this Play Station.
